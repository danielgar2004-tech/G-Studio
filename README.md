# G-Studio — Portafolio Web Profesional

Sitio web oficial de **G-Studio**, el estudio de desarrollo web freelance de Daniel García (Cajamarca, Perú).

---

## Estructura del proyecto

```
g-studio/
├── index.html          Sitio completo (HTML + CSS + JS en un solo archivo)
├── robots.txt          Directivas para rastreadores
├── sitemap.xml         Mapa del sitio para buscadores
├── README.md           Este archivo
├── DEPLOY.md           Instrucciones de despliegue en Vercel
└── imagenes/
    ├── logo.webp / .png            Logo horizontal
    ├── isotipo.webp / .png         Isotipo (símbolo "G")
    ├── logo-full.webp / .png       Logo completo
    ├── og-image.jpg / .webp        Imagen Open Graph (1200×630)
    ├── favicon.ico                 Favicon multi-resolución
    ├── favicon-16/32/192.png       Favicons PNG
    └── apple-touch-icon.png        Ícono para dispositivos Apple
```

## Características técnicas

| Área            | Implementación                                                        |
|-----------------|-----------------------------------------------------------------------|
| Stack           | HTML5 semántico · CSS3 · JavaScript vanilla — sin dependencias pesadas |
| Tipografía      | Space Grotesk (títulos) + Inter (cuerpo), vía Google Fonts            |
| SEO             | Meta tags, Open Graph, Twitter Cards, JSON-LD, robots.txt, sitemap    |
| Responsividad   | Mobile-first · breakpoints 320 / 768 / 1024 / 1440 px                 |
| Animaciones     | Canvas (Glitch Grid + partículas), scroll reveal, contadores          |
| Accesibilidad   | Foco visible, ARIA, `prefers-reduced-motion`, contraste AA            |
| Conversión      | CTAs WhatsApp con prefill diferenciado por sección                    |
